# Game Visual Style References

This document provides factual descriptions of visual styles from reference games. These are purely observational notes for reference during art development.

---

## Mini Warriors / Mini Warriors Reborn

### Overview
- **Genre:** Strategy/tower defense mobile game
- **Perspective:** 2D side-view with horizontal battlefield
- **Art Approach:** Cartoon style with HD visual quality

### Visual Characteristics

**Art Style:**
- Delicate, vivid cartoon imagery
- Realistic proportions within cartoon aesthetic
- Dynamic lighting and cinematic combat animations
- High polish level with crisp HD graphics

**Color Palette:**
- Vibrant and highly saturated colors
- Element-based color differentiation (fire, nature, etc.)
- High contrast between units and battlefield
- Note: Some players report issues with excessive bloom and red tinge affecting visibility

**Battlefield Design:**
- Horizontal layout with clear territorial divisions
- Clean, simplified backgrounds
- Designed to support massive unit counts (up to 10,000 units)
- Environment serves as backdrop rather than focal point

**Unit Display:**
- Small-scale units that remain readable at high density
- Over 300 hero types and 32 warrior categories
- Clear silhouettes for each unit type
- Formation-based positioning (cavalry, infantry, archers, etc.)
- Visibility challenges emerge at extreme unit counts

**Background Structure:**
- Relatively flat battlefield design
- Minimal layering to prioritize foreground clarity
- Simple approach allows for high unit density

**UI Integration:**
- UI designed to not obstruct battlefield view
- Hero skills displayed with prominent visual effects
- Functionality prioritized over decorative elements

**Visual Appeal Factors:**
- Epic scale feeling from commanding large armies
- Readable despite chaos
- Approachable cartoon aesthetic
- Satisfying combat visual feedback
- Clear battlefield state at a glance

## Cult of the Lamb

### Overview
- **Genre:** Action roguelike with base-building elements
- **Perspective:** Top-down isometric view
- **Art Approach:** 2.5D construction using Unity 3D with sprite-based characters

### Visual Characteristics

**Art Style:**
- Cartoonish quality with isometric/top-down perspective
- "Sunny pastoral horror" aesthetic theme
- Faux-3D structures created with sprite layering
- Evolved away from "Saturday morning cartoon" toward grounded look

**Color Palette:**

Primary colors:
- Earth tones: Deep browns (#5c3d1f), sepia (#a35c3e), cream (#f5c75c)
- Accent colors: Deep reds (#69000b, #c70000), purples (#be2797, #6a1256)
- Contrast strategy: Vibrant pops against muted base colors

Color philosophy:
- Pastel earth tones for base environment
- Vibrant colors for important interactive elements
- Strategic use of gold/yellow for highlights and attention
- Colors intentionally stand out against backgrounds

**Environment Design:**
- Top-down isometric perspective with fixed camera
- Room-based layout (dungeons, healing rooms, combat areas)
- 2.5D layering through sprite stacking and parallax
- Moderate detail level - atmospheric without overwhelming
- Natural, grounded aesthetic with cult-themed elements (runes, symbols)

**Character Display:**
- Cute animal characters with simple, readable designs
- Strong, instantly recognizable silhouettes
- Larger character size relative to environment compared to RTS games
- Smooth animation with clear attack patterns
- Clear combat telegraphing and invincibility frames during dodge
- Fixed camera can sometimes cause foreground object occlusion

**Depth and Layering:**
- 2.5D construction with sprites arranged in 3D space
- Parallax scrolling for environmental depth
- Clear foreground/midground/background separation
- Important elements typically placed toward top/back of screen

**UI Style:**
- Clean, minimal UI maintaining aesthetic consistency
- Integrated icons with consistent art style
- Color-coded information for quick readability
- Cartoonish but functional approach

**Polish Level:**
- High animation quality at smooth framerates
- Strategic use of particle effects and shaders
- Deliberate simplicity in some areas for clarity
- Strong focus on character silhouettes
- "Less is more" philosophy for visual design

**Visual Appeal Factors:**
- Juxtaposition of cute aesthetic with dark themes
- Emotional accessibility through adorable character design
- Cohesive aesthetic across all elements
- Readable combat with clear visual feedback
- Atmospheric immersion through 2.5D depth
- Original iconography and mythology creating unique identity

## Common Observations

### Clarity Principles
- Both games prioritize readability despite different scales
- Strong silhouettes used for instant recognition
- Clean backgrounds that support rather than compete with gameplay elements
- Strategic color use for differentiation and visual hierarchy

### Stylization Approach
- Neither attempts photorealism
- Stylization enables clearer visual communication
- Simplification improves both performance and readability

### Color Systems
- Mini Warriors: High saturation with high contrast
- Cult of the Lamb: Muted base with vibrant accents
- Both use color strategically for visual hierarchy

### Detail Management
- Mini Warriors: Minimal per-unit detail due to massive scale
- Cult of the Lamb: Moderate detail appropriate for closer camera
- General principle: Detail level inversely proportional to unit count

### Visual Feedback
- Both feature dynamic effects for combat feedback
- High-quality animation prioritized for game feel
- Strategic use of particles and shaders (not excessive)

## Notes on Tower Defense Visual Design

### Industry Observations

**Glanceability:**
- Instant battlefield comprehension at a glance
- Visual archetypes with distinct roles
- Status information via icons rather than text

**2D vs 3D Considerations:**
- 2D allows small details to stand out at final resolution
- Avoids asset bloat from multiple viewing angles
- Better performance with high unit counts

**Visual Hierarchy:**
- Important information always visible
- Detailed information accessible via interaction
- Avoiding constant information clutter in viewport

**Unit Design:**
- Silhouette should convey function
- Visual design supports quick identification
- Clear spacing between units to prevent visual overlap
